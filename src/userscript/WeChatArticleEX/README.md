## 微信推送浏览功能拓展

[脚本主页](https://github.com/ZiuChen/userscript)

### ✨ 主要功能：

* **悬停快速预览/复制/跳转文章封面图**
* **正文显示摘要，支持单击文本复制**
* 设置默认展示时间格式
* 隐藏了右侧悬浮的引导关注栏
* 在网络中搜索当前文章
* `ESC`临时显示/隐藏菜单
* 所有功能可独立配置是否启用

### 🔨 效果演示：

确保脚本在脚本管理器中处于启用状态，在浏览器中随意打开一篇推送，例：[春节期间小程序及小游戏审核调整通知](https://mp.weixin.qq.com/s/SuBSIEg13yhOYQoZFiL_iQ)。

*注：文章链接须以mp.weixin.qq.com开头*
